# AXI4-Lite Watchdog Timer

A lightweight **AXI4-Lite Watchdog Timer Controller** implemented in **Verilog RTL** and verified using **SystemVerilog**.

---

## Overview

This project implements an AXI4-Lite slave-based watchdog timer capable of monitoring system activity and generating a timeout event if the watchdog is not periodically refreshed.

---

## Features

* AXI4-Lite Slave Interface
* Programmable Timeout Counter
* Watchdog Kick Support
* Timeout Detection
* System Reset Generation
* Directed Verification Environment
